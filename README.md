# Operating Systems Project
Bu projede kendi linux shell programımızı yazmamız gerekiyor. Bu yazdığımız programımıza özelkendi komutlar olacaktır. Bu komutların da yine kendine özel çıktıları, yanıtları ve hata kodları olacaktır. Bizden istenen sistem programlama kapsamında bir C program yazmamızdır.

//
In this project, we need to write our own Linux shell program. This program we write will have its own custom commands. These commands will also have their own specific outputs, responses, and error codes. We are required to write a C program within the scope of system programming.
